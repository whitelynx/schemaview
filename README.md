schemaview
==========

Standalone JSON Schema viewer


Setup
-----

```bash
npm i
```


Running in the repository
-------------------------

```bash
./schemaview /path/to/file.schema.json
```


Installing globally
-------------------

```bash
ln -s $PWD/schemaview /usr/local/bin/
```

Then, to run:

```bash
schemaview /path/to/file.schema.json
```
